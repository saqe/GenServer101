# GenServer101

**A simple app to understand GenServer and it's core features.**

## Installation

### Clone the repo, install dependencies using
```shell 
mix deps.get  
```


```elixir
def deps do
  [
    {:genserver101, "~> 0.1.0"}
  ]
end
```


# EXECUTION
To execute the code. 
```shell
iex -S mix
```
Terminal will be started.

iex(1)>  `Client.perform_task()`
